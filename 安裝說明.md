# 美股掃圖 - 安裝說明

## 前置需求

- Python 3.11
- Git
- 網路連線

---

## Windows 安裝步驟

```bat
git clone https://github.com/uelliot88/scan-US.git
cd scan-US
python -m venv venv
venv\Scripts\pip install -r requirements.txt
```

更新資料並開啟網頁：

```bat
update_and_open.bat
```

---

## macOS / Linux 安裝步驟

```bash
git clone https://github.com/uelliot88/scan-US.git
cd scan-US
python3 -m venv venv
venv/bin/pip install -r requirements.txt
chmod +x update_and_open.sh
./update_and_open.sh
```

---

## 本地執行網頁

Windows:

```bat
venv\Scripts\streamlit run app.py
```

macOS / Linux:

```bash
venv/bin/streamlit run app.py
```

---

## 資料來源

- 股票 universe：Nasdaq screener 免費端點
- 價量資料：Yahoo Finance via `yfinance`
- UI/資料參考：`https://usstockflow.aihost.dev/`

`stock_concepts.json` 目前保留為本地補充主題快取；若沒有美股主題資料，網頁會改顯示 Nasdaq sector / industry。

---

## 測試用快速更新

完整美股 universe 下載時間較長。開發測試可限制前 N 檔：

Windows PowerShell:

```powershell
$env:MAX_STOCKS="200"; venv\Scripts\python update_data.py
```

macOS / Linux:

```bash
MAX_STOCKS=200 venv/bin/python update_data.py
```
