# 台股掃圖 — 安裝說明

## 前置需求

- Python 3.11
- Git
- 網路連線
- GitHub repo 推送權限

---

## Windows 安裝步驟

### 1. 下載專案

```bat
git clone https://github.com/uelliot88/scan-TW.git
cd scan-TW
```

### 2. 建立虛擬環境並安裝套件

```bat
python -m venv venv
venv\Scripts\pip install -r requirements.txt
```

### 3. 更新資料並開啟網頁

直接雙擊 `update_and_open.bat`，或在命令列執行：

```bat
update_and_open.bat
```

腳本會自動：

1. 若本機有私有市場主題更新工具，會更新 `stock_concepts.json`；否則沿用既有快取
2. 下載最新股票資料並產生 `uptrend_results.json`（約 10 分鐘）
3. 推送 `uptrend_results.json` 與 `stock_concepts.json` 到 GitHub
4. 開啟 Streamlit Cloud 網頁

---

## macOS / Linux 安裝步驟

### 1. 下載專案

```bash
git clone https://github.com/uelliot88/scan-TW.git
cd scan-TW
```

### 2. 建立虛擬環境並安裝套件

```bash
python3 -m venv venv
venv/bin/pip install -r requirements.txt
```

### 3. 設定執行權限（首次一次）

```bash
chmod +x update_and_open.sh
```

### 4. 更新資料並開啟網頁

```bash
./update_and_open.sh
```

腳本會自動：

1. 若本機有私有市場主題更新工具，會更新 `stock_concepts.json`；否則沿用既有快取
2. 下載最新股票資料並產生 `uptrend_results.json`（約 10 分鐘）
3. 推送 `uptrend_results.json` 與 `stock_concepts.json` 到 GitHub
4. 開啟 Streamlit Cloud 網頁

---

## 本地執行網頁

Windows:

```bat
venv\Scripts\streamlit run app.py
```

macOS / Linux:

```bash
venv/bin/streamlit run app.py
```

---

## 注意事項

- Streamlit Cloud 讀取的是 GitHub 上的檔案，推送後通常需等待約 1～2 分鐘才會更新。
- 若 GitHub 推送失敗，本地 `uptrend_results.json` 與 `stock_concepts.json` 仍會更新，但雲端網頁不會同步。
- `stock_concepts.json` 是市場主題本地快取；網頁顯示時只讀本地檔案，不會每次開頁都連外抓取。
- `stock_notes.json` 可用來人工補充主要營業項目；若不存在，網頁仍可正常執行。
