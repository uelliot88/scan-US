# 美股掃圖 - 安裝與使用說明

## 系統需求

- Python 3.11
- Git
- 可連線到 Nasdaq、Yahoo Finance、USStockFlow

## Windows 安裝

```bat
git clone https://github.com/uelliot88/scan-US.git
cd scan-US
python -m venv venv
venv\Scripts\pip install -r requirements.txt
```

更新資料、推送 GitHub 並開啟本機 Streamlit：

```bat
update_and_open.bat
```

## macOS / Linux 安裝

```bash
git clone https://github.com/uelliot88/scan-US.git
cd scan-US
python3 -m venv venv
venv/bin/pip install -r requirements.txt
chmod +x update_and_open.sh
./update_and_open.sh
```

## 只開啟本機 App

Windows:

```bat
venv\Scripts\streamlit run app.py
```

macOS / Linux:

```bash
venv/bin/streamlit run app.py
```

## 資料來源與篩選範圍

- 股票 universe：Nasdaq screener。
- 排除項目：ADR、ETF、基金、權證、特別股、信託、債券等非普通股標的。
- 價格資料：Yahoo Finance via `yfinance`。
- 市場主題：USStockFlow `https://usstockflow.aihost.dev/`。

`update_data.py` 會產生：

- `uptrend_results.json`：本專案掃出的型態、RS、成交量、sector、industry 等資料。
- `stock_concepts.json`：USStockFlow 主題資料、全市場主題強弱排名、標的與主題對應。

## App 功能摘要

- 可用「型態選擇」與「產業類別」篩選標的。
- 「今日領漲族群」與「今日領跌族群」來自 USStockFlow 全市場主題排名。
- 點選主題區塊後，會顯示該主題與本專案掃出標的的交集。
- 收藏勾選可跨頁累積，不論切換到哪一頁都可下載。
- TradingView 匯出檔會依細分產業分組，例如：

```txt
###Semiconductors###
AAOI,NVDA,AMD

###Aluminum###
AA
```

## 限制更新檔數

測試時可用 `MAX_STOCKS` 限制更新檔數，避免全市場更新耗時太久。

Windows PowerShell:

```powershell
$env:MAX_STOCKS="200"; venv\Scripts\python update_data.py
```

macOS / Linux:

```bash
MAX_STOCKS=200 venv/bin/python update_data.py
```
